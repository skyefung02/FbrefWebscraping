The portion of the data that is specific to FPL is from the vaastav/Fantasy-Premier-League repo. For the code to work, you need to clone this repo into your own local folder 

att_test.csv - This is the csv file which contains the test data for the attackers
att_explore_original.csv - This is the csv file which contains the training data for the attackers
att_finaldat.csv - This is the csv file which contains all of the data for the attackers
def_finaldat.csv - This is the csv file which contains all of the data for the defenders
